# Projeto do Bootcamp Santander 2025 - Ciência de Dados com Python

O dashboard desenvolvido transforma dados brutos de 296 assinaturas Xbox Game Pass em informações visuais para análise de vendas e add-ons.

## Funcionalidades do Dashboard

Dashboard Excel com análise de vendas de assinaturas Xbox Game Pass (Ultimate, Standard, Core) incluindo:

### Métricas Principais
- Receita EA Play Season Pass
- Receita Minecraft Season Pass
- Total Subscriptions Game Pass

### Interatividade
- Slicer por Subscription Type (Anual/Mensal/Trimestral)
- Atualização dinâmica dos valores
